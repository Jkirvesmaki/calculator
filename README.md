# Simple Calculator made with React and css

![calcimg](https://i.gyazo.com/2e683a463b33974ce16d82eeb56464b9.png)

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Start Server:

`npm start`  

To Visit App:

`localhost:3000`
